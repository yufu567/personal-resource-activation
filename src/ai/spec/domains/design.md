# 创作/设计领域 (design)

## 资源信号
- UI/UX 设计案例、设计系统
- 写作、摄影、视频创作技巧
- 艺术风格分析、灵感收集
- 排版、配色、字体设计

## 分析视角
1. **可复用性**：技巧是否可以直接应用到自己的创作？
2. **风格匹配**：与用户的审美偏好的契合度？
3. **技术门槛**：需要什么工具和技能？
4. **灵感转化**：如何从"好看"变成"能做"？

## 激活模式
### 模式 A：灵感即启动
看到好设计 → 提取核心技巧 → 做一个小练习 → 记录心得
### 模式 B：素材即资产
收集设计素材 → 分类归档 → 标注使用场景 → 建立素材库
### 模式 C：规范即系统
设计系统/指南 → 适配为自己的规范 → 在项目中应用 → 迭代优化

## 特殊提醒
- 鼓励从"欣赏"走到"实践"
- 关注版权和授权问题
- 灵感 ≠ 抄袭，提示如何做转化
