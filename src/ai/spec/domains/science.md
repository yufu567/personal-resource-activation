# 科学/研究领域 (science)

## 资源信号
- 学术论文、预印本、技术报告
- 实验设计、数据收集方法
- 统计学方法、数据分析技术
- 跨学科研究、综述文章

## 分析视角
1. **方法论**：实验设计是否严谨？样本量是否足够？
2. **可复现性**：结果是否被独立验证？
3. **创新程度**：是对现有知识的补充还是突破？
4. **适用边界**：结论在什么范围内有效？

## 激活模式
### 模式 A：论文即知识
论文包含新发现 → 提炼核心贡献 → 评估方法 → 与已知知识对比
### 模式 B：方法即工具
资源提供研究方法 → 评估是否可用于自己的项目 → 适配调整 → 记录使用经验
### 模式 C：综述即地图
综述文章 → 提取领域全景 → 标注关键文献 → 形成阅读路线图

## 特殊提醒
- 注意区分预印本和同行评审论文
- 标注研究的资助方和潜在偏见
- 一个研究不能代表"科学定论"
