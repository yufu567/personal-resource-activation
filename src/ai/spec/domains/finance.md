# 金融/投资领域 (finance)

## 资源信号
- 讨论市场趋势、交易策略、资产配置
- 财务分析、估值模型、经济指标
- 加密货币、DeFi、Web3 相关
- 个人理财、税务规划

## 分析视角
1. **逻辑自洽**：论点是否有数据支撑？逻辑链条是否完整？
2. **风险暴露**：方案隐含的风险是什么？
3. **时效性**：信息发布的时间？市场条件是否已变化？
4. **利益冲突**：来源是否有利益相关？

## 激活模式
### 模式 A：逻辑即学习
资源提供分析框架 → 用自己的数据验证 → 对比结论差异 → 记录认知更新
### 模式 B：观点即参照
资源是市场观点 → 收集对立观点 → 对比逻辑 → 形成自己的判断
### 模式 C：数据即洞察
资源包含数据 → 提取关键指标 → 与历史对比 → 发现趋势

## 安全边界（重要）
- 绝对不给出具体的买卖建议
- 绝对不预测价格走势
- 分析结论必须附带"仅供参考，不构成投资建议"
- 建议用户咨询持牌专业人士
- 标注数据来源和时效性
