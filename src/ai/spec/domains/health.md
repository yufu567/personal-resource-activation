# 健康/医疗领域 (health)

## 资源信号
- 讨论疾病、症状、治疗方案
- 营养学、运动科学、心理健康
- 医学研究、临床试验结果
- 养生、保健、生活方式

## 分析视角
1. **证据强度**：来自同行评审研究？个人经验？商业推广？
2. **安全性**：是否有已知风险或副作用？
3. **个体差异**：是否考虑了不同人群的适用性？
4. **来源资质**：作者是否有相关专业背景？

## 激活模式
### 模式 A：信息整理
资源提供健康信息 → 整理关键要点 → 标注证据等级 → 形成参考笔记
### 模式 B：习惯即行动
资源建议生活方式改变 → 拆解为小步骤 → 设定追踪指标 → 记录执行情况
### 模式 C：对比即决策
多个冲突观点 → 对比证据和方法 → 标注分歧点 → 咨询专业人士

## 安全边界（极其重要）
- 绝对不提供诊断
- 绝对不推荐或反对特定药物
- 绝对不替代专业医疗建议
- 所有输出必须包含："此分析不构成医疗建议，请咨询执业医师"
- 不处理用户的个人健康数据
- 心理健康类：标注危机热线信息
